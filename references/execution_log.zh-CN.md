# 执行日志模板

[English](execution_log.md) | [中文](execution_log.zh-CN.md)

每次生成报告期间维护 `results/execution_log.md`，避免最后复盘只靠记忆。

必备章节：

1. 智能体名称、可见模型名称、平台、skill 路径和日期。
2. 用户提示词时间线。
3. 已收集或推断的输入。
4. 读取、创建、修改以及刻意未修改的文件。
5. 执行命令、工作目录、关键输出、退出状态和生成文件。
6. 失败项、根因、修复过程和是否完全解决。
7. 项目、前端、截图、配图、DOCX、PDF、参考文献和打包验证结果。
8. 限制和真实降级说明。
9. 最终交付物。
10. 对 skill 的改进建议。

不得记录 API Key、私密 token 或不必要的个人信息。
